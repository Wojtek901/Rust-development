To send data from powershell use: Invoke-RestMethod -Uri "http://127.0.0.1:3000/items" -Method Post -Body '{"name": "", "is_purchased": false}' -ContentType "application/json"
To send data on linux use: curl -X POST http://127.0.0.1:3000/items -H "Content-Type: application/json" -d '{"name": "", "is_purchased": false}'
See data in yours browser under link: http://127.0.0.1:3000/items
Or in power shell: Invoke-RestMethod -Uri "http://127.0.0.1:3000/items" -Method Get
Or with curl: curl.exe http://127.0.0.1:3000/items
